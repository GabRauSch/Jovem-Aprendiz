Desafio 1 - Porta errada!

Olá, como está você? Nesse desafio vamos fazer algo bastante simples e que faz uma diferença gingantesca no que diz respeito ao seu nivel como programadora!

O problema: A porta no nosso servidor está errada! ajeite isso e torne as coisas mais faceis para a vida dos programadores que vao usar seu codigo no futuro!
Dicas:
O problema se encontra no server.ts
Ao fazer o deploy do projeto, a nossa porta muda, temos que arranjar um método para que seja possivel não precisar reescrever nenhum código quando formos por no ar.
não basta mudar o console log ou a porta para resolver o problema... lembre que se esse problema ocorreu uma vez, algum outro programador pode vir e cometê-lo novamente
utilize uma biblioteca chamada dotenv, dentro do arquivo, sete a porta para 3000
dentro do arquivo .env, que deve ficar na raiz do projeto, ponha:
    PORT=3000
no seu server.ts escreva:
import dotenv from 'dotenv'; dotenv.config(); server.listen(process.env.PORT)
